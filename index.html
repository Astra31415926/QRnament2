QRnament

Lab v2.5 // Ritual Scan

How to Use

Enter text to generate an RGB ornament. The Blue channel holds the data (QR), while Red and Green channels hold 8-fold symmetric art. Scan a file to perform the Extraction Ritual.

Technology

Uses qrcode-generator for encoding, jsQR for decoding, and Web Audio API for synthesized scan sounds.

Author

Designed for experimental data visualization.
